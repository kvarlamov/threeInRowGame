# threeInRowGame

Игра 3 в ряд.

### Описание
Игрок вводит координаты соседних клеток в формате A1B1.

### Запуск

0. Перейти в папку ThreeInRowGame
1. Выполнить команду:
~~~bash
dotnet build
~~~
2. Запустить игру командой:
~~~bash
dotnet run
~~~

### Требования
.NET 8.0 => [download](https://dotnet.microsoft.com/ru-ru/download/dotnet/8.0)
